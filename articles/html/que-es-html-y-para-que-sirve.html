Que es HTML y para que sirve?

Que es el HTML?

HTML es (en resumen) el componente más básico de la Web, este define el significado y la estructura del contenido web. Además de HTML, generalmente se utilizan otras tecnologías para describir la apariencia/presentación de una página web (CSS) o la funcionalidad/comportamiento (JavaScript).

Esta pagina esta hecha con HTML, CSS y JavaScript

HTML utiliza "etiquetas" que pueden contener texto, imágenes y otro contenido para mostrarlo en un navegador Web. Las marcas HTML incluyen "elementos" especiales como < head >, < title >, < body >, < header >, < footer >, < article >, < section >, < p >, < div >, < span >, < img >, < aside >, < audio >, < canvas >, < datalist >, < details >, < embed >, < nav >, < output >, < progress >, < video >, < ul >, < ol >, < li > y muchos otros.

Para que sirve el HTML?

HTML no es un lenguaje de programación; es un lenguaje de marcado que define la estructura de tu contenido. HTML consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera.

Las etiquetas de encierre pueden hacer de una palabra o una imagen un hipervínculo a otro sitio, se pueden cambiar palabras a cursiva, agrandar o achicar la letra, etc.
